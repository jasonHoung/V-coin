# Platform Control Layer

## 模組功能

平台控制層是 V-Coin 系統的核心，負責：
- 任務調度與分配
- GPU 節點管理
- 驗證系統
- 信譽管理
- 獎勵結算

## 技術棧

- **語言**: Go 或 Rust
- **資料庫**: PostgreSQL（主要資料）+ Redis（快取、佇列）
- **訊息隊列**: RabbitMQ / Kafka
- **API**: RESTful + WebSocket

## 核心模組

### 1. 任務調度系統
- 接收使用者推論/訓練請求
- 根據 GPU 需求選擇合適節點
- 管理任務優先級佇列
- 處理任務重試與故障轉移

### 2. 節點管理
- 節點註冊與驗證
- 心跳監控
- GPU 狀態追蹤
- 節點上下線管理

### 3. 驗證系統
- 任務抽樣驗證（10%）
- Hidden Validation（5%）
- 結果比對與相似度計算
- 驗證失敗處理

### 4. 信譽系統
- 信譽分數計算
- 成功/失敗記錄
- 動態驗證率調整
- 節點封禁與恢復

### 5. 獎勵結算
- 計算任務獎勵
- 呼叫智能合約鑄造 Coin
- 記錄獎勵歷史
- Slashing 執行
